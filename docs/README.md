## 🎯 按场景查找

### 我想部署系统

1. 阅读 [系统安装配置手册](./系统安装配置手册.md)
2. 按步骤安装Python环境和依赖
3. 配置 `configs/config.yaml`
4. 运行系统并验证
5. 如果使用Docker，参考Docker部署章节

### 我想集成到PLC/机器人

1. 阅读 [API接口使用手册 - TCP接口](./API接口使用手册.md#tcp接口)
2. 了解 `catch` 命令的使用
3. 参考PLC集成示例代码
4. 测试TCP连接和数据传输
5. 实现错误处理和重试逻辑

### 我想开发上位机软件

1. 阅读 [API接口使用手册 - MQTT接口](./API接口使用手册.md#mqtt接口)
2. 了解所有MQTT命令
3. 参考上位机监控示例代码
4. 实现配置管理、模型测试等功能

### 我需要执行标定

1. 阅读 [标定操作手册](./标定操作手册.md)
2. 制作或准备标定板
3. 按照流程执行12点标定
4. 验证标定精度（RMSE<3mm为优秀）
5. 如果精度不满足，参考常见问题章节

### 系统出现问题

1. 查看 [故障排查手册 - 快速诊断](./故障排查手册.md#快速诊断)
2. 根据症状找到对应的问题章节
3. 按照解决方案逐步排查
4. 查看日志文件 `logs/VisionCorePro_*.log`
5. 如果无法解决，准备诊断报告并联系支持

## 📋 配置文件参考

完整的配置文件说明请参考: [系统安装配置手册 - 配置文件详解](./系统安装配置手册.md#配置文件详解)

### 最小化配置（用于快速测试）

```yaml
logging:
  enable: true
  level: INFO

camera:
  enable: true
  connection:
    ip: 192.168.2.99        # 修改为实际相机IP
    port: 2122

model:
  backend: auto
  path: models/seasoning_11.18.pt  # 修改为实际模型
  conf_threshold: 0.7

DetectionServer:
  enable: true
  host: 0.0.0.0
  port: 8888

mqtt:
  enable: false

sftp:
  enable: false

roi:
  enable: true
  minArea: 3000
```

## 🛠️ 命令速查

### 系统管理

```bash
# 启动系统
python -m app.main

# 启动为系统服务（Linux）
sudo systemctl start visioncore

# 查看日志
tail -f logs/VisionCorePro_$(date +%Y-%m-%d).log
```

### 网络诊断

```bash
# 测试相机连接
ping 192.168.2.99
telnet 192.168.2.99 2122

# 测试TCP服务器
telnet localhost 8888
```

### 性能监控

```bash
# 查看进程状态
ps aux | grep python | grep main.py

# 查看端口占用
netstat -tuln | grep 8888
```

更多命令请参考: [故障排查手册 - 附录：常用命令速查](./故障排查手册.md#附录常用命令速查)
